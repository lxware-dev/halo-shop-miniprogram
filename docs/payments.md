# 登录与支付

本文介绍**登录能力**、**微信支付接入前置条件**、**订单与物流**、**法务链接**等关键业务环节。

> [!NOTE]
> 前端发起的请求会携带销售渠道标识（如 `X-Sales-Channel: MINI_PROGRAM`）。商户号、API 密钥、支付回调、分账等均在**服务端与微信商户平台**配置，需由 Halo / 运维人员完成。

## 登录能力

### 配置位置

在 `src/config/app.config.json` 或 `app.config.local.json` 的 `auth.loginMethods` 字段中配置：

```json
{
  "auth": {
    "loginMethods": {
      "primary": "haloAccount",
      "supported": ["haloAccount", "phoneQuick"]
    }
  }
}
```

完整的配置字段说明见 [config.md](./config.md#authloginmethods)。

### 手机号一键登录（`phoneQuick`）

- 仅在**微信小程序环境**下完整可用，需在真机或开发者工具中验证。
- 用户通常需先同意用户协议与隐私条款（以实际页面交互逻辑为准）。
- 若 Halo 后台未开通对应绑定/登录能力，将出现登录失败或无法获取手机号的问题。

### 账号密码登录（`haloAccount`）

- 依赖 Halo 侧用户体系与接口，请确保与后台配置的认证方式一致。

### 开发调试：Basic Auth

`dev.basicAuth` 仅在**开发模式**下用于联调，配置方式见 [config.md](./config.md)。

> [!WARNING]
> 请勿在生产环境依赖 Basic Auth 作为正式登录方案。

## 微信支付

### 职责划分

| 端            | 职责                                                   |
| ------------- | ------------------------------------------------------ |
| 小程序前端    | 拉起微信支付控件、展示支付结果页、订单列表与详情展示   |
| Halo / 服务端 | 商户号配置、API 密钥、创建订单、签名、支付回调、分账等 |

### 接入前检查清单

- [ ] 微信小程序与微信商户号已完成**关联**（按微信当前规则完成绑定）。
- [ ] Halo / Halo Pro 侧已配置**小程序支付**相关参数（具体以 Halo 官方文档为准）。
- [ ] 小程序使用 **HTTPS** 合法域名访问 Halo（见 [prepare-go-live.md](./prepare-go-live.md)）。
- [ ] 已使用**体验版 + 真机**完整走通：下单 → 调起支付 → 查询订单结果。

### 常见支付失败原因

| 现象                   | 可能原因                                           |
| ---------------------- | -------------------------------------------------- |
| 无法调起支付控件       | 订单未创建成功、服务端未返回支付参数、商户配置错误 |
| 提示权限或配置错误     | AppID / 商户号不匹配、小程序未关联商户             |
| 支付成功但订单状态未变 | 异步通知失败、Halo 回调 URL 配置错误或被防火墙拦截 |

> [!TIP]
> 以上问题需结合**服务端日志**与**微信商户平台**排查。前端可记录请求失败时的错误提示与订单号，便于定位问题。

## 订单与物流

- 订单列表、详情、确认收货、物流追踪等功能依赖 Halo 返回的数据结构，后台需正确维护订单状态与物流单号。
- 若商品为虚拟商品或服务类，需在后台与业务规则中明确标记**不需要物流**，避免前端展示异常。

## 法务链接配置

在 `business.legalDocuments` 中配置各类协议与法务文档的链接（见 [config.md](./config.md)）：

| 字段               | 说明             |
| ------------------ | ---------------- |
| `userAgreement`    | 用户协议         |
| `privacyPolicy`    | 隐私政策         |
| `paymentAgreement` | 支付说明（按需） |
| `platformRules`    | 平台规则（按需） |
| `qualification`    | 经营资质（按需） |

> [!IMPORTANT]
> 上线前，所有链接应为 **HTTPS** 可访问地址。若使用 `<web-view>` 打开协议页，对应域名需在微信公众平台配置**业务域名**。

## 帮助中心与客服

- **帮助中心 FAQ**：通过 `business.helpCenterFaqs` 配置问答列表，详见 [config.md](./config.md)。
- **联系客服**：`business.contactServiceEnabled` 控制是否展示客服入口；实际客服能力（企业微信、微信客服消息等）需按微信与 Halo 后台能力配置。
